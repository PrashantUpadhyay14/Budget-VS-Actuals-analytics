# 📊 Sales Performance Dashboard | Microsoft Excel

## 📌 Project Overview

This interactive Microsoft Excel dashboard provides a comprehensive analysis of sales performance by comparing Actual Sales, Budget Plan, and Previous Year (PY) results. The dashboard enables users to evaluate sales trends, monitor business performance, and identify variances across multiple locations using dynamic filters and visualizations.

---

# 🎯 Business Objective

The objective of this dashboard is to compare actual sales performance against planned targets and previous year results, helping businesses monitor growth, identify performance gaps, and support data-driven decision-making.

---

# 📊 Dashboard KPIs

- 💰 Actual Sales
- 📈 Budget Plan
- 📊 Previous Year (PY) Sales
- 📉 Year-over-Year (YoY) Growth %
- 📉 Plan Variance %

---

# 📈 Analysis Performed

## 1️⃣ Sales Performance Analysis

- Actual Sales by city
- Budget Plan comparison
- Previous Year comparison
- Sales performance evaluation

---

## 2️⃣ Actual vs Plan Analysis

Compare actual sales against planned targets to identify overperforming and underperforming locations.

---

## 3️⃣ Year-over-Year Growth Analysis

Measure annual sales growth by comparing current year performance with the previous year.

---

## 4️⃣ Plan Variance Analysis

Evaluate the percentage difference between actual sales and planned sales to assess target achievement.

---

## 5️⃣ Regional Performance Analysis

Analyze sales performance across multiple cities including:

- Atlanta
- Brooklyn
- Detroit
- Madison
- Miami
- New York City
- Portland
- San Diego
- San Jose
- Seattle

---

## 6️⃣ Interactive Filtering

Users can dynamically analyze data using:

- Year Filter
- Month Filter

---

# 💡 Key Business Insights

- Compare Actual Sales against Budget Plan for every city.
- Identify regions exceeding or missing sales targets.
- Track Year-over-Year sales growth trends.
- Highlight positive and negative plan variances.
- Enable quick monthly and yearly performance monitoring through interactive filters.
